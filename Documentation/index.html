About

Google Calendar Script

This script allows to speed up the creation of events in Google Calendar, add a video meeting and notify the participants and instructor by Gmail. Through a Google Sheets with an established format.

Tutorial

Quick Steps

  1. Create a calendar in Google Calendar
  2. Download the Sheet format
  3. Fill the Sheet correctly
  4. Click on Calendar / Send Invites
  5. Wait a few moments depending on the number of events
  6. If all elements are created, the job is done
  7. If not, check the status column if the status is failed, in the error description column there is a brief explanation
  8. check all failed states and repeat from step 4
